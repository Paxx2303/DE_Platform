**Ghi chú:**
- Các câu hỏi được thiết kế sát với các chỉ số trong file SNI ISO 37122:2019.
- Phân loại truy vấn bao gồm: Truy vấn số liệu, So sánh, Xu hướng, Top/Bottom, Lọc theo thời gian/khu vực, Giải thích dữ liệu.
- ESG: **E** = Environment, **S** = Social, **G** = Governance.

File này có thể dùng để xây dựng chatbot Smart City thông minh.
